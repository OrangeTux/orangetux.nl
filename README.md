# orangetux.nl

## Installation

Clone the sources:

```
$ git clone git@github.com:OrangeTux/orangetux.nl.git
$ cd orangetux.nl
```

Get the theme:

```
$ mkdir themes
$ git clone https://github.com/dillonzq/LoveIt.git themes/LoveIt
```

Run it:

```
$ hugo server --theme=hugo-steam-theme --buildDrafts -w
```
